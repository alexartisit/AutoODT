Double-click UNINSTALL_Office_AutoRunAdmin.bat -> click "YES"(for Administrator rights) -> (will open uninstall Office) click Uninstall, 
(wait for finish) -> click "Close" button on Office window after finish -> (system wil be restarted) -> after logon
AFTER LogOn double-click INSTALL_Office_AutoRunAdmin.bat -> click "YES"(for Administrator rights) -> 
this will automatically install office with the configuration from the configuration.xml file
BE CAREFUL CONFIGURATION.XML FROM FROJECT IS A DUMMY FILE REPLACE WITH YOUR CONFIGURATION
